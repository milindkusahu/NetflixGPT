# Netflix GPT

Netflix clone with GPT-3 integration.

### Features

1. Login/Sign Up

   * Sign In /Sign up Form
   * Redirect to Browse Page
2. Browse (After authentication)

   * Header
   * Main Movie
     * Trailer in Background
     * Title & Description
     * Movie Suggestions
       * Movie List * N
3. NetflixGPT

   * Search Bar
   * Movie Suggestions
